# UnitySVC Labs

UnitySVC Labs is the platform's own provider, offering free and experimental services to demonstrate gateway capabilities.

## Public S3 Datasets

These services proxy publicly available S3 datasets through the UnitySVC S3 storage gateway. They require no seller credentials (anonymous public bucket access) and are priced at $0.

See the [repository README](../../README.md) for the full list of available datasets.
